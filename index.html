TrueCrypt

Free open-source disk encryption software for Windows 7/Vista/XP, Mac OS X, and Linux. Is it audited yet?

  • Creates a virtual encrypted disk within a file and mounts it as a real disk.

  • Encrypts an entire partition or storage device such as USB flash drive or hard drive.

  • Can encrypt a partition or drive where Windows is installed (pre-boot authentication).

  • Provides plausible deniability, in case an adversary forces you to reveal the password (hidden volume or operating system).

How to: create volumes, mount volumes, use hidden volumes.

FileVault

Provided by Apple on Mac operating systems, under System Preferences → Security & Privacy.

  • FileVault 1: available in OS X versions 10.3 to 10.6. Encrypts a user's home directory.

  • FileVault 2: OS X Lion (10.7) and later. Full disk, XTS-AES 128 encryption.

BitLocker

Provided by Microsoft on Windows 7 Ultimate and Enterprise Edition, and Windows 8 Pro and Enterprise under Control Panel → System and Security

Gpg4win

Includes a tool called GNU Privacy Assistant with an interface that can encrypt, decrypt, sign and verify files.

LUKS

Linux Unified Key Setup

  • Standard for Linux hard disk encryption.

  • Included in the installer of many modern distributions.

  • Cryptsetup utility supports many types of encrypted volumes.

Example:

$ apt-get install cryptsetup
$ cryptsetup luksFormat /dev/sda1
$ cryptsetup luksOpen /dev/sda1 data
$ mkfs.ext4 /dev/mapper/data
$ mount -t ext4 /dev/mapper/data /media/data
$ update-initramfs -u

Also check out eCryptfs.
